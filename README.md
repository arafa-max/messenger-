# Messenger Lite

![Bun](https://img.shields.io/badge/Bun.js-latest-black?style=flat-square&logo=bun)
![WebSocket](https://img.shields.io/badge/WebSocket-realtime-blue?style=flat-square)
![Railway](https://img.shields.io/badge/Deployed-Railway-purple?style=flat-square)

A fast, lightweight real-time messaging app built with Bun.js and WebSockets.

🔗 **Live Demo:** [messenger-lite.up.railway.app](https://messenger-lite.up.railway.app/)
📖 **API Docs:** [messenger-lite.up.railway.app/swagger](https://messenger-lite.up.railway.app/swagger)

## Features
- ⚡ Real-time messaging with WebSockets
- 🚀 Ultra-fast Bun.js runtime
- 💬 Multiple chat rooms
- 📖 Swagger API docs

## Tech Stack
| Layer | Technology |
|-------|-----------|
| Runtime | Bun.js |
| Protocol | WebSocket |
| Deploy | Railway |

## Getting Started
```bash
bun install
bun run index.ts
```
